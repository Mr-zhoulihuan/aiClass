# 机器学习

## 一、什么是机器学习

**机器学习**（Machine Learning, ML）是人工智能的一个核心分支，指计算机通过数据**自动学习模式与规律**，而不需要人工显式编写每一条规则。简单说：**用数据训练模型，让模型对新数据做出预测或决策**。

### 1.1 与传统编程的对比

| | 传统编程 | 机器学习 |
|------|----------|----------|
| 输入 | 数据 + 规则 | 数据 + 答案（标注）或无标注数据 |
| 输出 | 结果 | 规则（模型） |
| 典型场景 | 计算器、业务逻辑 | 图像识别、推荐系统、语音助手 |

### 1.2 核心要素

要完成一个机器学习任务，通常需要三样东西：

1. **数据**（Data）：模型学习的原材料，质量决定模型上限
2. **模型**（Model）：一个可学习的数学结构（如线性回归、决策树、神经网络）
3. **优化算法**（Optimization）：通过损失函数衡量误差，迭代更新模型参数

## 二、机器学习的三大流派

根据训练数据是否有标签，机器学习分为三类：

### 2.1 监督学习（Supervised Learning）

训练数据包含**输入**和对应的**正确输出（标签）**，模型学习从输入到输出的映射。

- **回归**（Regression）：预测连续值 → 房价预测、气温预测
  - 典型算法：线性回归、岭回归、SVR
- **分类**（Classification）：预测离散类别 → 垃圾邮件检测、图像识别
  - 典型算法：逻辑回归、决策树、SVM、KNN、神经网络

> 监督学习是目前最成熟、应用最广的范式。

### 2.2 无监督学习（Unsupervised Learning）

训练数据**没有标签**，模型自己发现数据中的结构或模式。

- **聚类**（Clustering）：将相似样本自动分组 → 用户分群、图像压缩
  - 典型算法：K-Means、DBSCAN、层次聚类
- **降维**（Dimensionality Reduction）：压缩特征维度，保留主要信息 → 数据可视化、特征提取
  - 典型算法：PCA、t-SNE、UMAP

### 2.3 强化学习（Reinforcement Learning）

智能体（Agent）在环境中通过**试错**学习，根据获得的**奖励**信号调整策略。

- 典型场景：围棋（AlphaGo）、游戏 AI、机器人控制、自动驾驶
- 关键概念：**状态（State）→ 动作（Action）→ 奖励（Reward）→ 策略（Policy）**
- 典型算法：Q-Learning、DQN、PPO

## 三、核心概念

| 概念 | 说明 |
|------|------|
| **特征**（Feature） | 用于描述样本的属性，如房屋面积、颜色直方图 |
| **标签**（Label） | 监督学习中样本对应的真实结果 |
| **训练集 / 测试集** | 训练集用于训练模型，测试集用于评估泛化能力 |
| **过拟合**（Overfitting） | 模型在训练集上表现好，但对新数据表现差（学得太"死"） |
| **欠拟合**（Underfitting） | 模型连训练集都没学好，表达能力不够 |
| **损失函数**（Loss Function） | 衡量模型预测与真实值差距的函数 |
| **梯度下降**（Gradient Descent） | 最常用的优化方法，沿着梯度方向更新参数降低损失 |
| **交叉验证**（Cross Validation） | 将数据分成多份轮流训练/验证，更稳健地评估模型 |

## 四、典型算法速览

### 4.1 经典算法

| 算法 | 类型 | 核心思想 | 特点 |
|------|------|----------|------|
| **线性回归** | 回归 | 用直线拟合数据 | 简单、可解释强 |
| **逻辑回归** | 分类 | 用 Sigmoid 函数输出概率 | 二分类基线算法 |
| **决策树** | 分类/回归 | 通过条件判断做决策 | 可解释性好，易过拟合 |
| **随机森林** | 分类/回归 | 多棵决策树投票 | 抗过拟合，鲁棒性好 |
| **SVM**（支持向量机） | 分类 | 寻找最大间隔超平面 | 小样本效果好 |
| **K-Means** | 聚类 | 按距离将样本分为 K 簇 | 简单快速，需指定 K |
| **KNN**（K 近邻） | 分类/回归 | 根据最近邻居投票 | 无需训练，计算慢 |
| **朴素贝叶斯** | 分类 | 基于贝叶斯定理 + 特征独立假设 | 文本分类效果好 |

### 4.2 集成学习

将多个弱模型组合成一个强模型：

- **Bagging**（如随机森林）：并行训练多个模型，投票/平均
- **Boosting**（如 XGBoost、LightGBM）：串行训练，每个模型纠正上一个的错误
- **Stacking**：用多个模型的输出作为新模型的输入

> **XGBoost / LightGBM** 是 Kaggle 竞赛中最常用的表格数据神器，工业界应用极广。

## 五、机器学习的完整流程

```
问题定义 → 数据采集 → 数据清洗与预处理 → 特征工程 → 
模型选择与训练 → 模型评估 → 调参优化 → 部署上线 → 
监控与迭代
```

### 各步骤要点

1. **问题定义**：明确是分类、回归还是聚类？业务目标是什么？
2. **数据采集**：数据库、日志、API、爬虫等
3. **数据清洗**：处理缺失值、异常值、重复数据
4. **特征工程**（Feature Engineering）：
   - 数值型：归一化、标准化、离散化
   - 类别型：独热编码（One-hot）、标签编码
   - 文本型：TF-IDF、词嵌入
5. **模型选择**：简单问题用线性模型，复杂问题用树模型或神经网络
6. **训练与调参**：用验证集监控，使用网格搜索（Grid Search）或贝叶斯优化
7. **评估**：选择恰当指标（见下文）
8. **部署**：模型上线为 API 或嵌入应用

## 六、评估指标

### 6.1 分类任务

| 指标 | 含义 | 适用场景 |
|------|------|----------|
| **准确率（Accuracy）** | 预测正确的比例 | 类别均衡时 |
| **精确率（Precision）** | 预测为正类中真正为正的比例 | 垃圾邮件检测（误报敏感） |
| **召回率（Recall）** | 真实正类中被正确找出的比例 | 疾病筛查（漏报敏感） |
| **F1 分数** | 精确率和召回率的调和平均 | 类别不均衡的通用指标 |
| **AUC-ROC** | 模型区分正负类的能力 | 排序类问题 |
| **混淆矩阵** | 预测 vs 真实的交叉计数表 | 详细诊断模型错误 |

### 6.2 回归任务

| 指标 | 说明 |
|------|------|
| **MSE**（均方误差） | 预测值与真实值差的平方的平均，对异常值敏感 |
| **MAE**（平均绝对误差） | 预测值与真实值差的绝对值的平均，更鲁棒 |
| **R²**（决定系数） | 模型解释了多少比例的方差，越接近 1 越好 |

## 七、机器学习 vs 深度学习

| 对比维度 | 机器学习 | 深度学习 |
|----------|----------|----------|
| 特征工程 | 依赖人工设计和选择特征 | 自动从原始数据中学习特征 |
| 数据量 | 小到中等数据量即可 | 需要大量数据才能发挥优势 |
| 计算资源 | CPU 即可，低门槛 | 通常需要 GPU，算力需求高 |
| 可解释性 | 较高（如决策树、线性回归） | 较差（"黑盒"问题突出） |
| 适用数据 | 结构化表格数据为主 | 图像、文本、音频、视频等非结构化数据 |
| 代表算法 | 随机森林、XGBoost、SVM | CNN、RNN、Transformer |

> **实际工程中**：表格数据优先尝试 XGBoost/LightGBM；图像/文本/语音等非结构化数据优先考虑深度学习。

## 八、常见误区与注意事项

| 误区 | 正确做法 |
|------|----------|
| 数据越多模型一定越好 | 数据质量比数量更重要，垃圾进垃圾出 |
| 模型越复杂越好 | 优先从简单模型开始，复杂模型容易过拟合且难维护 |
| 训练集准确率高 = 好模型 | 用测试集评估，警惕过拟合 |
| 特征越多越好 | 冗余特征会引入噪声，需做特征选择 |
| 调参能解决一切 | 数据质量、特征工程往往比调参重要 10 倍 |
| 部署后就不管了 | 模型上线后需持续监控数据漂移和性能衰退 |

## 九、学习路径建议

1. **数学基础**：线性代数（矩阵运算）、概率统计（贝叶斯、分布）、微积分（梯度）
2. **入门课程**：吴恩达《Machine Learning Specialization》（Coursera 经典课程）
3. **动手实践**：Kaggle 上的入门竞赛（Titanic、House Prices）
4. **学习框架**：
   - **scikit-learn**：ML 经典算法全覆盖，API 简洁
   - **XGBoost / LightGBM**：竞赛和工业级表格数据神器
   - **Pandas + NumPy**：数据处理基本功
5. **深入进阶**：
   - 《统计学习方法》（李航）— 理论扎实
   - 《机器学习》（周志华，西瓜书）— 入门经典教材
   - 《Hands-On Machine Learning》（Aurélien Géron）— 实战导向
6. **持续跟进**：关注 Papers With Code、ArXiv、ML 领域的顶级会议（NeurIPS、ICML、KDD）
