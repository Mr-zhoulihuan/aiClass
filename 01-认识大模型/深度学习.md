# 深度学习

## 一、什么是深度学习

**深度学习**（Deep Learning, DL）是机器学习的一个子领域，它使用**多层人工神经网络**从原始数据中自动学习层次化的特征表示。"深"指的是网络中包含大量隐藏层，层数越多，模型能学习的抽象层次越高。

### 1.1 深度学习 vs 传统机器学习

| 对比维度 | 传统机器学习 | 深度学习 |
|----------|-------------|----------|
| **特征工程** | 人工设计特征（耗时、依赖专家经验） | 自动从数据中学习特征 |
| **数据量** | 小到中等数据量即可 | 需要大规模数据才能发挥优势 |
| **计算资源** | CPU 足够 | 依赖 GPU/TPU，算力需求高 |
| **可解释性** | 较高（决策树、线性模型） | 较差（"黑盒"问题） |
| **适用数据** | 结构化表格数据为主 | 图像、文本、音频、视频等非结构化数据 |
| **模型复杂度** | 参数量少，训练快 | 参数量大（百万到千亿），训练慢 |

### 1.2 为什么深度学习现在才火起来？

三个关键条件在 2010 年代陆续成熟：

1. **大数据**：互联网、传感器、移动设备产生了海量训练数据
2. **算力爆发**：GPU（NVIDIA CUDA）让并行矩阵运算提速数十倍
3. **算法突破**：ReLU 激活函数、Batch Normalization、Dropout、Adam 优化器、Transformer 等

## 二、神经网络基础

### 2.1 神经元（Neuron）

深度学习的基本单元是**人工神经元**，它模拟生物神经元的"接收 → 处理 → 输出"过程：

```
输入x₁ → 权重w₁
输入x₂ → 权重w₂  →  求和 Σ(wᵢ·xᵢ) + bias  →  激活函数 f  →  输出 y
输入x₃ → 权重w₃
```

### 2.2 常见激活函数

| 激活函数 | 公式 | 特点 | 适用场景 |
|----------|------|------|----------|
| **Sigmoid** | 1 / (1 + e⁻ˣ) | 输出范围 (0,1)，平滑 | 二分类输出层 |
| **Tanh** | (eˣ - e⁻ˣ) / (eˣ + e⁻ˣ) | 输出范围 (-1,1)，零中心 | 循环网络、早期全连接 |
| **ReLU** | max(0, x) | 计算快，缓解梯度消失 | **现代网络默认选择** |
| **Leaky ReLU** | max(αx, x) | 解决 ReLU 死亡问题 | 深层网络备选 |
| **Softmax** | eˣᵢ / Σeˣⱼ | 输出概率分布，总和为 1 | 多分类输出层 |
| **GELU** | x · Φ(x) | 平滑版 ReLU | Transformer（BERT、GPT） |

### 2.3 前馈传播（Forward Propagation）

数据从输入层逐层向前传递，每一层对上一层的输出做**线性变换 + 激活函数**，直到输出层得到预测结果。

### 2.4 反向传播（Backpropagation）

核心训练算法：

1. 前向计算预测值和损失
2. 利用**链式法则**从输出层向输入层逐层计算梯度
3. 使用**梯度下降**更新每层的权重和偏置

> 反向传播 + 梯度下降是深度学习训练的"发动机"。

## 三、主流网络架构

### 3.1 卷积神经网络（CNN）

专门处理**网格结构数据**（如图像）的网络。

| 核心层 | 作用 |
|--------|------|
| **卷积层**（Convolution） | 用卷积核扫描图像提取局部特征（边缘、纹理、形状） |
| **池化层**（Pooling） | 下采样，降低特征图尺寸，减少参数量 |
| **全连接层** | 将提取的高层特征映射到最终分类结果 |

- **代表模型**：LeNet、AlexNet、VGG、ResNet、EfficientNet
- **典型应用**：图像分类、目标检测（YOLO）、图像分割（U-Net）

> **ResNet** 的"残差连接"（Skip Connection）解决了深层网络退化问题，是近年最关键的架构创新之一。

### 3.2 循环神经网络（RNN）

专门处理**序列数据**（文本、时间序列、音频）的网络。

- **核心思想**：隐藏状态在时间步之间传递，形成"记忆"
- **LSTM**（长短期记忆）：通过遗忘门、输入门、输出门缓解长期依赖问题
- **GRU**（门控循环单元）：LSTM 的简化版，参数量更少

> 在 NLP 领域，RNN 系列已基本被 **Transformer** 取代，但在时序预测中仍有应用。

### 3.3 Transformer

**当前最重要的深度学习架构**，几乎统治了 NLP 并正在扩展到 CV、多模态等领域。

核心创新：

| 组件 | 作用 |
|------|------|
| **自注意力**（Self-Attention） | 每个 token 都能直接关注所有其他 token，捕捉长距离依赖 |
| **多头注意力**（Multi-Head Attention） | 多组注意力并行学习不同维度的关系 |
| **位置编码**（Positional Encoding） | 给模型提供序列中词语的位置信息 |
| **Feed-Forward Network** | 每个位置的独立非线性变换 |
| **层归一化 + 残差连接** | 稳定训练，防止梯度消失 |

- **代表模型**：BERT、GPT 系列、ViT（视觉 Transformer）、Swin Transformer
- **典型应用**：机器翻译、文本生成、图像分类、多模态理解

### 3.4 生成对抗网络（GAN）

两个网络相互博弈：

- **生成器**（Generator）：尝试生成"以假乱真"的数据
- **判别器**（Discriminator）：尝试区分真实数据和生成数据

- **应用**：图像生成、图像超分辨率、风格迁移
- **代表模型**：DCGAN、StyleGAN、CycleGAN

### 3.5 扩散模型（Diffusion Models）

目前最先进的图像生成技术（Stable Diffusion、DALL·E、Midjourney 的基础）：

1. **前向过程**：逐步向图像添加噪声，直到变成纯噪声
2. **反向过程**：学习从噪声中逐步去噪还原出图像

> 扩散模型在图像生成质量上超越了 GAN，成为当前主流。

## 四、训练技巧与优化

| 技术 | 作用 |
|------|------|
| **梯度下降变体** | SGD、Momentum、Adam（最常用）、AdamW |
| **学习率调度** | 余弦退火、指数衰减、Warm-up |
| **Batch Normalization** | 每层输入归一化，加速训练，缓解梯度消失 |
| **Dropout** | 训练时随机丢弃部分神经元，防止过拟合 |
| **数据增强** | 旋转、翻转、裁剪、颜色抖动，扩大有效数据量 |
| **正则化**（L1/L2） | 限制权重大小，防止过拟合 |
| **早停**（Early Stopping） | 验证集不提升时停止训练 |
| **梯度裁剪**（Gradient Clipping） | 防止梯度爆炸 |
| **混合精度训练**（FP16） | 速度翻倍，显存减半 |
| **分布式训练** | 数据并行、模型并行、流水线并行 |

## 七、局限与挑战

| 挑战 | 说明 |
|------|------|
| **数据饥渴** | 需要大量标注数据，标注成本高 |
| **算力昂贵** | 训练大模型耗电巨大，碳排放问题受关注 |
| **黑盒问题** | 模型决策过程难以解释，在医疗、金融等领域是障碍 |
| **过拟合风险** | 参数远多于样本数时容易过拟合（大模型相对更好） |
| **对抗攻击** | 微小扰动就能让模型做出完全错误的判断 |
| **灾难性遗忘** | 学习新任务时可能忘记旧任务（在增量学习中突出） |
| **部署困难** | 大模型体积大、推理慢，边缘设备上部署有挑战 |
| **数据偏见** | 训练数据中的偏见会被模型放大和固化 |

## 八、深度学习的主要研究方向

1. **大语言模型（LLM）**：更大规模、更长上下文、更强推理
2. **多模态学习**：文本+图像+音频+视频的统一理解与生成
3. **自监督学习**：无需人工标注，利用数据本身的结构学习
4. **模型压缩与加速**：量化、蒸馏、剪枝、MoE（混合专家）
5. **可解释 AI（XAI）**：让模型决策过程透明可理解
6. **联邦学习**：数据不出本地，协同训练模型，保护隐私
7. **强化学习 + 深度学习**：从游戏 AI 到机器人控制、自动驾驶
8. **AI for Science**：用深度学习加速科学发现（蛋白质、材料、天气）

## 九、学习路径建议

### 9.1 数学基础

- **线性代数**：矩阵运算、特征值分解、SVD
- **微积分**：偏导数、链式法则、梯度
- **概率统计**：分布、贝叶斯、最大似然估计

### 9.2 入门课程

1. **吴恩达《Deep Learning Specialization》** — 最经典的深度学习入门课（Coursera）
2. **李沐《动手学深度学习》** — 配套 PyTorch/MXNet 实战代码，中英文都可用
3. **Stanford CS231n**（计算机视觉）— 经典 CNN 课程
4. **Stanford CS224n**（自然语言处理）— NLP + Transformer 入门

### 9.3 经典书籍

- 《深度学习》（Ian Goodfellow 等，"花书"）— 理论大全
- 《动手学深度学习》（李沐等，d2l.ai）— 理论与实战结合
- 《Deep Learning with Python》（François Chollet）— Keras 实战

### 9.5 进阶资源

- **论文**：关注 NeurIPS、ICML、ICLR、CVPR、ACL 等顶会
